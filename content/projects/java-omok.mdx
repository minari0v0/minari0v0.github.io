## 프로젝트 개요
객체 지향 프로그래밍(OOP)의 원리를 실제로 적용해보기 위해 자바(Java)로 개발한 오목 게임입니다.

## 주요 기능
- **GUI 인터페이스**: 직관적인 바둑판 디자인과 돌 놓기 애니메이션
- **승리 판정 알고리즘**: 3x3 금지 룰(선택 사항) 및 오목 완성 시 승리 알림
- **게임 초기화**: 언제든 게임을 다시 시작할 수 있는 리셋 기능

## 개발 후기
- GUI 이벤트 처리(MouseListener)에 대한 이해도 상승
- 승리 조건을 체크하는 알고리즘을 직접 구현하며 논리적 사고력 향상
